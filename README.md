## 🚀 التشغيل المحلي

```bash

npm run dev
```

## 🏗️ البناء

```bash
npm run build
```

## 📦 التقنيات المستخدمة

- React 18 + TypeScript
- Vite (bundler)
- Tailwind CSS
- Recharts (الرسوم البيانية)
- Lucide React (الأيقونات)
- localStorage (التخزين المحلي)
